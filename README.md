# Touch-Portal-Axis-Cam-PTZ-CamStreamer-Plugin
A [Touch Portal](https://www.touch-portal.com/) plugin that controls Axis IP cameras and CamStreamer apps directly over the LAN — the Touch Portal sibling of the Stream Deck *Axis Cam + CamStreamer* plugin in the parent folder. It talks straight to the camera (VAPIX + CamStreamer/CamOverlay/CamSwitcher CGIs)
